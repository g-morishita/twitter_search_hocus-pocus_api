Twitter Search Hocus-Pocus API

This is an implementation of two API endpoints that connects through the twitter API. The one of them is to get the list of tweets with the given hashtag. The other is to get the list of tweets that user has on his feed in json format.

Contact the developer

Apache2.0

Schemes
HTTP

hashtags

operations with hashtags

GET

/hashtags/{hashtag}

Find tweets by a hashtag

Returns the list of tweets with a given hashtag in a JSON format, which is the same format as what you get using a Twitter Search API. The details is here

Parameters

Name Description
hashtag required hashtag by which you find tweets
pages_limit the number of pages to retrieve. One page has 100 tweets

users

operations with users

GET

/users/{user}

Find tweets by a user

Returns the list of tweets that certain user has on his/her own feed in a JSON format, which is the same format as what you get using Twitter Search API. The details is here

Parameters

Name Description
userrequired User on whose feed you find tweets
pages_limit the number of pages to retrieve. One page has 100 tweets
